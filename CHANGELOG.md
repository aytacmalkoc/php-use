# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Initial release of PHP Use
- `php-use.bat` script for switching between PHP versions
- `php.bat` wrapper for Windows Command Prompt
- `php` wrapper script for Git Bash support
- Automatic PATH management on first run
- Support for multiple PHP versions (7.3, 8.2, 8.3, 8.4)
- Complete documentation in README.md
- MIT License

### Features
- Easy PHP version switching with a single command
- Multi-shell support (Windows Command Prompt & Git Bash)
- Composer compatibility
- Clean project structure with organized `bin/` directory
